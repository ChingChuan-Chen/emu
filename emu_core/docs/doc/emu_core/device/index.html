[−][src]Module emu_core::device

The lowest-level, core functionality for controlling a GPU device

You can use this module in isolation from the rest of Emu for the most fine control over the internals of the library. The most important things in this module that are likely of interest to you are Device, DeviceBox<T> , and DeviceFnMut.
